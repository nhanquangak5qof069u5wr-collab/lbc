# 比赛演示脚本

## 阶段一：正常联邦训练

- 讲解 8 个节点分别代表不同医疗机构。
- 强调原始数据不出本地，只上传模型更新。
- 点击“开始训练”，观察首页精度曲线稳步上升。

## 阶段二：开启恶意攻击

- 切到“攻击控制”页。
- 选择“标签翻转”或“后门攻击”。
- 设置 2 个恶意节点，开启攻击。
- 返回总览页，说明恶意节点已经开始污染全局模型。

## 阶段三：开启动态防御

- 切到“防御分析”页，开启动态防御。
- 讲解异常分布、相似度热力图、信誉分变化和权重下调。
- 指出恶意节点被降权或隔离后，全局模型开始恢复。

## 阶段四：展示结果对比

- 切到“结果对比”页。
- 对比无防御与动态防御下的准确率、损失、后门成功率。
- 强调系统具备识别、抑制、恢复的完整闭环。

## 阶段五：导出实验报告

- 切到“实验报告”页。
- 点击导出 Markdown。
- 展示生成的实验摘要，作为比赛材料的一部分。
